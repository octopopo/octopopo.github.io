Water Simulation

Project

In "Water Simulation", we use the techique of environment mapping to set up the skybox. Then we implemented a surface that can reflect the enviroment above, and refract the objects below. Since the whole application was powered rendered by GPU, so our surface can interact with real-time object, like a video. We intergrated OpenCV to allow our project interact with webcam, so that we can tested our application with real-time environment.

My Role

Below are my works:

1. Set up the OpenGL frame work: Implemented the frame work of the project that included glew and glut. So that all teammate can develop their feature quickly.

2. Intergrated OpenCV into project: Intergrated OpenCV into our project. With OpenCV, we can use webcam to capture the real-time image and stored them into our buffer.
